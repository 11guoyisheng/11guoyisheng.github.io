      无序列表 — 常用

      在 HTML 中，无序列表使用

        标签来创建。每个列表项使用
      • 标签。与有序列表不同，无序列表不会自动为每个列表项添加编号，而是使用项目符号（通常是圆点）来标记每个项。

        可以进行嵌套

        以下是

          标签的一些关键点：
          默认项目符号: 默认情况下，无序列表使用圆点作为项目符号。
          样式属性: 可以通过 list-style-type CSS 属性更改项目符号的样式，例如：
          disc: 圆点（默认）。
          circle: 空心圆。
          square: 方块。
          none: 无项目符号。

          无序列表在 HTML 中用于表示没有特定顺序的项目集合。以下是一些常见的应用场景：
          导航菜单: 无序列表常用于网站的导航菜单，因为菜单项通常没有特定的顺序。
          特性列表: 用于列出产品或服务的特性或优点。
          任务列表: 用于表示待办事项或任务列表，尤其是在不需要按顺序完成任务时。
          项目符号列表: 在文档中用于强调或列出要点。
          社交媒体链接: 用于列出社交媒体平台的链接。
